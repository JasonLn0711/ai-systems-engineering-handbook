# Module 02: Cloud, On-Prem, And Private Cloud

## Purpose

Teach deployment environments and why enterprises choose public cloud, private cloud, on-prem, or hybrid models.

## Core Questions

- What is the difference between public cloud, private cloud, on-prem, and hybrid deployment?
- Why does on-prem not automatically mean safer?
- What operational cost appears when AI models move into customer-controlled environments?

## Suggested Chapters

- Public Cloud, Private Cloud, And On-Prem.
- Hybrid Cloud And Enterprise Boundaries.
- Bare Metal, VM, Container, And K8s.
- VPC, VPN, Subnet, Firewall, And Load Balancer.
- Data Residency And Compliance.
- On-Prem AI Deployment Lifecycle.
- Private Cloud AI Appliance Pattern.

## First Labs

- Design an on-prem AI deployment flow.
- Draw a hybrid cloud architecture with data and governance boundaries.
