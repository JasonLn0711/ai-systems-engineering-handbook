# Module Spec: Cloud, On-Prem, And Private Cloud

## Purpose

Teach deployment environment choices and their effects on data boundary, security, cost, latency, ownership, and maintenance.

## Learning Outcomes

- Compare public cloud, private cloud, on-prem, and hybrid deployment.
- Explain why on-prem is not automatically safer.
- Design a basic on-prem AI deployment lifecycle.

- `01-public-cloud-private-cloud-onprem.md`
- `02-hybrid-cloud-and-enterprise-boundaries.md`
- `06-onprem-ai-deployment-lifecycle.md`
