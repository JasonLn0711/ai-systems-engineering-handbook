# Module 00: AI Systems Foundations

## Purpose

Teach what makes an AI application become an AI system.

## Core Questions

- What is the difference between a model, an application, and a system?
- How does one AI request move through product, data, model, policy, logging, and evaluation layers?
- Why is enterprise AI a systems problem rather than a model-only problem?
