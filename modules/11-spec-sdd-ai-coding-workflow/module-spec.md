# Module Spec: Spec, SDD, And AI-Assisted Coding Workflow

## Purpose

Teach how to use Codex and AI agents without losing architecture, module boundaries, data contracts, testing, or reviewability.

## Learning Outcomes

- Write product specs and SDDs before implementation.
- Define module boundaries and interface contracts.
- Review AI-generated code for coupling, context pollution, hidden dependencies, and missing tests.
