# Module Spec: RAG And Data Pipeline

## Purpose

Teach enterprise knowledge ingestion, metadata, retrieval, reranking, context construction, SQL + KB integration, and evaluation.

## Learning Outcomes

- Explain RAG as a data pipeline.
- Design ingestion, parsing, chunking, metadata, embeddings, reranking, context construction, citations, and evals.
- Identify stale knowledge, hallucination, and missing-context risks.

## Required First Chapters

- `01-rag-mental-model.md`
- `04-chunking-strategies.md`
- `05-metadata-generation.md`
