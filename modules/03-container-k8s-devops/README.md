# Module 03: Container, K8s, And DevOps

## Purpose

Teach packaging, orchestration, rollout, and deployment debugging for AI services.

## Core Questions

- Why do containers matter for AI services?
- What does Kubernetes add beyond Docker?
- How do health checks, config, secrets, services, ingress, and GPU scheduling affect AI workloads?
- How do you debug failed deployments?

## Suggested Chapters

- Why Container.
- Dockerfile, Image, Container, Volume, And Network.
- Docker Compose For AI Services.
- K8s Mental Model.
- Pod, Deployment, Service, And Ingress.
- ConfigMap, Secret, And Health Check.
- GPU Workloads In K8s.
- Deployment Debugging.
- CI/CD For AI Services.

## First Labs

- Containerize a FastAPI AI service.
- Deploy an AI service to K8s.
- Debug CrashLoopBackOff.
