# Module Spec: Container, K8s, And DevOps

## Purpose

Teach packaging, orchestration, rollout, health checks, and deployment debugging for AI services.

## Learning Outcomes

- Explain Docker image, container, runtime, volume, network, and Compose.
- Explain Kubernetes pod, deployment, service, ingress, config, secret, and health check.
- Debug common deployment failures.

## Required First Chapters

- `01-why-container.md`
- `02-dockerfile-image-container-volume-network.md`
- `04-k8s-mental-model.md`
