# Module Spec: AI Gateway And Agent Governance

## Purpose

Teach scalable control of agents, tools, knowledge bases, SQL sources, skills, adapters, policies, memory, approval gates, and audit logs.

## Learning Outcomes

- Explain why enterprise AI needs an AI Gateway.
- Design MCP server, skill, adapter, policy engine, registry, memory governance, and audit-log boundaries.
- Identify cross-agent data leakage and privilege escalation risks.

- `01-why-ai-gateway-exists.md`
- `03-mcp-server-mental-model.md`
- `04-skill-and-adapter-architecture.md`
