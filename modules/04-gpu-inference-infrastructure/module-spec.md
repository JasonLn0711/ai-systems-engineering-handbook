# Module Spec: GPU And AI Inference Infrastructure

## Purpose

Teach AI inference compute planning for local and enterprise deployments.

## Learning Outcomes

- Estimate VRAM from model weights, quantization, KV cache, context length, batch size, and concurrency.
- Compare local GPU, hosted compute, vLLM, Ollama, CUDA, and ROCm trade-offs.
- Explain multi-model scheduling and capacity planning.

## Required First Chapters

- `01-ai-inference-compute-mental-model.md`
- `02-gpu-vram-model-weights-activation-kv-cache.md`
- `04-vllm-ollama-and-model-serving.md`
