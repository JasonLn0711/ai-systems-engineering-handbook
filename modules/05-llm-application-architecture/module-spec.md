# Module Spec: LLM Application Architecture

## Purpose

Teach how prompt-based prototypes become maintainable AI services.

## Learning Outcomes

- Design service boundaries around prompts, tools, structured output, retries, fallbacks, streaming, logs, and evals.
- Explain where model behavior ends and application architecture begins.

## Required First Chapters

- `01-from-prompt-to-ai-service.md`
- `02-api-service-router-gateway.md`
- `03-prompt-template-and-structured-output.md`
