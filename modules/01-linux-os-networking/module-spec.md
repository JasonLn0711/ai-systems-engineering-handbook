# Module Spec: Linux, OS, And Networking

## Purpose

Teach the operating-system and network foundations needed to deploy and debug AI services in local, on-prem, and customer-controlled environments.

## Learning Outcomes

- Explain process, service, daemon, systemd, port, socket, DNS, TLS, SSH, firewall, user, permission, and log basics.
- Debug common local deployment and on-prem connectivity failures.

## Required First Chapters

- `01-linux-mental-model.md`
- `02-process-service-daemon-systemd.md`
- `03-filesystem-permission-user-group.md`
