# Case Study — Semiconductor Audio Event Detection

## Scenario

半導體產線需要偵測異常聲音、遠場收音、降噪、環境聲音特徵萃取，以及可能的人聲事件，例如怒罵或危險提示。

## Key Questions

1. 收音設備與距離如何影響可行性？
2. 是否需要降噪、source separation、VAD、audio event classification？
3. 是否需要 real-time alert？
4. 如何定義 false positive / false negative 的成本？
5. 如何做現場資料收集與標註？
