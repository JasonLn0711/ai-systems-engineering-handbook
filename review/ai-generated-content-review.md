# AI-Generated Content Review

Use this checklist for any AI-generated chapter, lab, case study, glossary entry, or prompt.

- [ ] The output follows the correct template.
- [ ] Claims are supported or written as bounded tutorial guidance.
- [ ] No raw private source content is copied.
- [ ] Terms are consistent with the module glossary.
- [ ] Security and governance sections are present.
- [ ] Failure modes are concrete.
- [ ] Exercises or verification steps exist.
- [ ] Cross-links are plausible.
- [ ] The content does not rewrite repo taxonomy.
