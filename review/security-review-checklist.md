# Security Review Checklist

- [ ] IAM boundary identified.
- [ ] Data boundary identified.
- [ ] PII handling identified.
- [ ] Prompt injection considered.
- [ ] Tool abuse considered.
- [ ] Memory leakage considered.
- [ ] RAG exfiltration considered.
- [ ] Audit log completeness considered.
- [ ] Red-team tests proposed.
- [ ] Incident recovery path identified.
